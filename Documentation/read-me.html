2Product Dev Cycle

After review of the requirements for this project it was agreed upon collectively that we would come up with an Application that was something that we would use, or was needed. We wanted to create a product that could be viable to be built out longer term. There are bike rental Ford and Jump(Uber owned)companies, motorized scooter companies, Lyft, Uber, but there are no Scooter companies, Kick Scooters that is! And ScootWee was born! A kickscooter rental service for the urban jungle!

So, we began our Psuedo coding, laying out our steps, db, we would take to meet MVP for approval by the powers (Justin).

A kickscooter rental service for the City at high density areas of mass transit for those who are not comfortable with the bike lanes and traffic and can be safe on sidewalks! A simple $5 a day fee, log in, see where there are scooters near you, reserve, pay and pick it up! Goals to work with major companies like Razor, Micro, Xooter. And try to work with Ford in a joint share of space.

4CSS Structure

We adapted this template which includes 5 css files. Utilizing the information learned in class (poem)on Tuesday in how to set up CSS to move objects in Ruby. Here is a list of the stylesheet files:

  • bootstrap.css - standard bootstrap css styles.
  • style.css - common css styles for this template.
  • style-magnific-popup - contain css styles for the lightbox gallery.
  • animate.css- a cross-browser library of CSS animations
  • fonts.css - base styles for the Fontastic icon fonts.
